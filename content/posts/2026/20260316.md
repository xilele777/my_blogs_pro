## 如何提交更改？

```html
1.查看修改状态 git status 
2.暂存所有修改（新增/修改/删除） git add .
3.提交（写清晰的提交信息） git commit -m "feat: 新增2026文章/修复配置"
4.推送到远程 git push origin main
```
